# KinalAPP

Este proyecto es básicamente una API que hice con 
Spring Boot para manejar clientes. La idea es que 
se pueda registrar gente, buscarlos, actualizarlos 
y borrarlos usando peticiones desde cualquier 
aplicación.

La aplicación también sirve para que otros programas
o sistemas puedan conectarse fácilmente y obtener 
información de clientes sin tener que manejar 
la base de datos directamente.

## Requisitos Previos
* **Java 21**
* **SpringBoot**
* **Maven**
* **MySQL**
* **Intellij IDEA**

## Cómo se Instala y se ejecuta
* **Paso 1:**
Ir a google y buscar Intellij IDEA
* **Paso 2:**
Descargar e instalar Intellij IDEA
* **Paso 3**
Ejecutar Intellij IDEA
* **Paso 4:**
Descargar el archivo 
* **Paso 5:**
Ejecutar el programa KinalAPP
